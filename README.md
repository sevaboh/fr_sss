# fr_sss
Solver for 1D fractional model of poroelasticity

fr_soil_strain_stress.cpp - solver used in https://doi.org/10.1007/s40324-024-00369-1
fr_soil_strain_stress_v2.cpp - its modified version
fr_soil_strain_stress_particle_transport.cpp - solver where 1D fractional model of poroelasticity is combined with sand,silt,clay particle transport, soil density and water retention curves recalculation.
